# R223M-P4-P1 默认视图与审核视图规则

| 维度 | 默认教师阅读稿 | 审核 / 开发视图 |
| --- | --- | --- |
| 小教判断 | 一句教师关注 | 核心判断 / 高风险 / 建议动作 / 教师确认点 |
