P4-P1 不再把“素材/组件确认”重复压到每个环节正文前，而是汇总为本课级确认门。教师确认前，这些内容均不写入正式备课本。

| 类型 | 待教师确认项 |
| --- | --- |
| 大屏素材 | 毛笔工序图 + 笔头/笔杆/笔尾圈画；四个为什么 + 结构圈画 + 关键词；设计案例细节圈画 + 握/放/收/找方向；一件只是装饰的作品 + 一件解决防滑/防滚/易找问题的作品；盘的技法、线材/纸材/形状材料半成品、材料过多与合适材料对比；结构、功能、材料、使用、方便、牢固、舒服、有创意词云；赠笔语句式 + 《一年级购买文具建议书》《文具课堂使用指南》 |
| 学习单栏位 | 我记住的毛笔工序 / 它让毛笔更好地……；我选择的为什么 / 我的猜想 / 我从哪里看出来；我想解决的小麻烦 / 我想到的小办法；我想解决的问题 / 我用了什么材料 / 它带来的改变；我选择的材料 / 我使用的方法 / 它解决的问题 / 我还想继续改的地方；我的新朋友原来……我用了……现在可以…… / 我收到的建议；我把它送给……因为它可以…… / 这给我的购买或使用建议是…… |
| 组件候选 | circle_and_annotate；match_cards；compare_two_images；sentence_starter；photo_submit；learning_sheet_record；technique_step_demo；material_choice_board；keyword_feedback；gallery_walk |
| 评价证据 | 发现文具结构与使用关系；能观察并说明文具结构与使用关系；能说明设计理由；能和小伙伴一起合作完成学习任务；我能发现新材料特性、新使用技法；我能在展示会中自信表达自己的想法；展示会中自信表达自己的想法 |

## 操作规则

- 可确认：确认某个大屏、学习单栏位、组件候选或评价证据进入后续预览。
- 可修改：调整素材、话术、学习单栏位或评价口径。
- 可替换：用替代组件或替代证据承接同一教学责任。
- 可删除：删除非必要派生物；若删除唯一证据，必须先补替代证据。
